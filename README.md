# Project 1

## About
This project is a full-stack CRUD (Create, Read, Update, Delete) application developed using Angular for the frontend and Node.js + Express + PostgreSQL for the backend. It allows users to perform standard operations on data with a clean and responsive user interface.

---

## Features
- Create, Read, Update, and Delete records
- Fully responsive Angular frontend
- RESTful API using Node.js and Express
- PostgreSQL database integration
- Clear project structure for easy maintenance

---

## Technologies Used
- **Frontend:** Angular, HTML, CSS, TypeScript
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Version Control:** Git & GitHub
